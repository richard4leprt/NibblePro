# NibblePro

**Free CAM software for turret punch press machines**

NibblePro generates NC programs for CNC turret punch presses. It reads DXF geometry and produces machine-ready NC code with full toolpath control.

## Features

- DXF import (lines, arcs, circles, blocks)
- Punch, nibble line, nibble arc, corner notch, hole grid operations
- Revolver (turret) tool management with up to 32 stations
- Multi-part sheet layout
- NC code export with repositioning (G25/G27)
- NC simulation with step-by-step animation
- Undo/Redo
- Interface languages: Polish, English, German
